# fleet

System for orchestrating local fleet of autonomous vehicles.

## Installation

In order to install Go, you must install the appropriate package from the following link:

    https://golang.org/dl/

Set your `$GOPATH` if you haven't already.
```
export GOPATH=$HOME/go
```

Pull the repository from Github.
```
cd $GOPATH/src
go get github.com/sdpetrides/fleet
```
